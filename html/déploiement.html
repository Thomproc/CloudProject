Déploiement de l'application Web

Afin de déployer ce site web, j'ai d'abord créé une Web App dans le portail Azure et un dépôt Git public afin de stocker le code source final.
Par la suite, j'ai développé localement le site jusqu'à ce que le rendu visuel soit satisfaisant, puis j'ai publié l'entièreté du code dans le dépôt Git créé précédemment.
À ce stade du projet, il restait à établir le lien entre la Web App et les fichiers sources. Pour cette dernière étape, je me suis rendu sur ma Web App dans le portail Azure, puis dans l'onglet "Centre de déploiement" où il est possible de déployer le code source stocké dans GitHub de deux manières :
  • Automatiquement en associant son compte GitHub à la Web App ("GitHub")
  • Manuellement en fournissant l'URL du dépôt ("GIT externe")
